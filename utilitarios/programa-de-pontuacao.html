Programa de Pontuação - Hotel Real

Como funciona

O sistema de pontuação do Hotel Real foi criado para valorizar e recompensar clientes e colaboradores conforme suas interações com a plataforma.

A cada reserva confirmada, os usuários acumulam pontos de forma automática. A quantidade de pontos gerados é baseada em três fatores:

  • O valor total da reserva
  • O tipo de usuário (cliente, cliente VIP, colaborador...)
  • A suíte escolhida (cada uma possui um fator de conversão)

Por exemplo, uma reserva na Suíte Master pode gerar mais pontos do que uma Suíte Luxo, mesmo com valores similares, dependendo do fator configurado.

Como os pontos são computados

Após a finalização da reserva e o pagamento via Tap to Pay, o sistema recebe um retorno via webhook contendo os dados da transação. Com base nisso:

  • A reserva é validada automaticamente
  • O sistema verifica qual suíte foi reservada
  • É aplicada a configuração de pontuação correspondente
  • Os pontos são adicionados ao perfil do usuário

Tudo ocorre de forma segura e automática, sem necessidade de ação manual.

Benefícios e resgates

Os pontos acumulados podem ser usados para resgatar benefícios, como upgrades de quarto, descontos, brindes ou serviços especiais. O resgate é feito diretamente pelo sistema com base nos benefícios disponíveis.
